Compatibility

We recommend using the latest version of back-end and front-end whenever possible. If you are interested in using an older version, see the below table to see which versions are compatible with each other.

Back-end Recommended front-end versions
2.0.0 Front-end 2.0.0
1.0.2 Front-end 1.0.3
1.0.1 Front-end 1.0.1
1.0.0 Front-end 1.0.1
